## 🚀 1. What is this App?
**Opus Pro** is an AI-powered SaaS platform that transforms long-form video content (podcasts, interviews, streams) into high-impact, viral short-form clips (TikTok, Reels, Shorts).

### Core Engine Capability:
- **Universal Import**: Downloads from YouTube, Twitch, and 1000+ sites.
- **AI Transcription**: Uses Groq Whisper for lightning-fast speech-to-text.
- **Viral Analysis**: Uses LLaMA 3.1 to identify the "hooks" and most shareable segments.
- **Dynamic Face-Tracking**: Automatically crops landscape video to 9:16 while keeping speakers centered.
- **Animated Captions**: Word-by-word "Karaoke style" captions with power-word highlighting.

## 🛠️ 3. Technical Implementation Details
- **Database**: Currently using SQLite with shared Docker volumes for persistence between the web server and workers.
- **State Management**: Redis handles the real-time progress bar updates via WebSockets.
- **Video Engine**: FFmpeg is used as the core "Video Shader" to burn captions and crop frames.

## 🚀 4. Future Upgrades (The Roadmap)

### 📈 Scaling & Speed
- **GPU Migration**: Move from CPU rendering to NVIDIA GPU (NVENC). This will make clipping **20x faster**.
- **PostgreSQL Migration**: Move from SQLite to a hosted Supabase Postgres for handling thousands of concurrent users.
- **Cloud Storage**: Switch `STORAGE_MODE` to `cloud` using AWS S3 or Cloudflare R2 for massive video storage.

### 🎭 AI Enhancements
- **Multi-Speaker Detection**: Automatically detect when two people are talking and create "Split-Screen" (Stack) layouts.
- **B-Roll Injection**: Use AI to automatically overlay stock footage or images based on the transcript topics.
- **AI Thumbnails**: Generate high-CTR YouTube/TikTok thumbnails for each clip using DALL-E 3 or Flux.

### 🎨 UI/UX Features
- **Caption Editor**: Let users change colors, fonts, and fix transcript typos in the browser.
- **Manual Crop**: Allow users to adjust the AI's crop if they want a different angle.
- **Mobile App**: Wrap the frontend in Capacitor/React Native for iOS and Android deployment.
